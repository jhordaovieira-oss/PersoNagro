# 🌱 PersonalAgro
### Plataforma de Inteligência Agrícola Personalizada

Projeto desenvolvido para o **Concurso Agrinho 2026**.

## O que é?
SaaS (Software como Serviço) voltado para produtores rurais brasileiros, oferecendo cotações diárias, análise de mercado com IA e relatórios automáticos via WhatsApp.

## Funcionalidades
- 📊 Dashboard com cotações em tempo real
- 📰 Radar da Semana com análise de notícias por IA
- 📈 Análise pessoal semanal com IA
- 💬 Relatórios automáticos via WhatsApp
- 🧮 Calculadora de ponto de equilíbrio
- ⚙️ Gestão de assinatura via Stripe

## Tecnologias
- React + Vite + Tailwind CSS
- Node.js + PostgreSQL
- Integração Z-API (WhatsApp)
- Stripe (pagamentos)
